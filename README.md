# Smart Support Ticket Frontend

Frontend application for the Smart Support Ticket System.

Provides a role-based interface for users, engineers, and admins to manage and track support tickets.


## Features

- User authentication
- Role-based dashboards
- Ticket creation and tracking
- Real-time assignment visibility
- Responsive UI


## Tech Stack

- React (Vite)
- Tailwind CSS
- Axios



## Routing

- Client-side routing using React Router
- Vercel rewrite configuration for SPA refresh support

## Deployment

- Hosted on Vercel
- check that out:- https://smart-support-ticket.vercel.app/
